Funcionalidades de la Aplicación

Legumbrería JM - Sistema de Comercio Electrónico

Una aplicación web completa desarrollada con arquitectura MVC que incluye gestión de usuarios, catálogo de productos, sistema de ventas, gestión empresarial avanzada, control financiero, gestión del personal, procesos operativos, seguridad robusta y notificaciones automáticas.

Gestión de Usuarios y Autenticación

Registro de Usuarios (RF-01)

  • Registro completo de nuevos clientes
  • Validaciones exhaustivas de campos:
    • Documento (8-10 dígitos)
    • Nombres completos (3-100 caracteres, solo letras)
    • Teléfono (exactamente 10 dígitos)
    • Correo electrónico válido
    • Contraseña segura (mínimo 8 caracteres, mayúscula, minúscula, número y símbolo)
    • Pregunta y respuesta de seguridad
  • Encriptación de contraseñas con bcrypt
  • Envío automático de correo de bienvenida
  • Generación automática de token JWT

Inicio de Sesión Seguro

  • Validación de credenciales
  • Sistema de intentos fallidos (máximo 3 intentos)
  • Bloqueo automático de cuentas tras múltiples fallos
  • Generación y gestión de tokens JWT
  • Envío de correo de confirmación de inicio de sesión
  • Verificación de tokens en base de datos

Gestión de Perfiles

  • Visualización del perfil del usuario autenticado
  • Edición de datos personales (nombres, teléfono, correo)
  • Envío de correo de confirmación al actualizar perfil
  • Validación de permisos por token
